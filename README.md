# Online European Portuguese — pilot

Static GitHub Pages website for a three-session European Portuguese pilot.

## Course access

The three session pages use the simple access code `351`. This is a lightweight browser-side gate for a pilot, not strong authentication.

## Publishing on GitHub

Upload all files and the `assets` folder to the root of the `main` branch. GitHub Pages will update automatically.

## Media

Audio buttons request a European Portuguese (`pt-PT`) voice from the visitor's browser. The exact voices available depend on the device. The video area in Session 1 is ready for an approved teaching clip; replace the placeholder with an embedded or local video when the final clip is selected.
